# 插件推荐

# 安装插件

参考：安装及使用教程。

可以在官方社区网站或 Figma 主页左侧的 Plugins 标签页浏览可供下载的插件库。

# 字体插件

# Better Font Picker

一款提供更好的字体搜索及预览的插件。

# Chinese Font Picker

中文字体选择器，选择和预览系统中已安装的中文字体，该插件是基于开源的 Better Font Picker 创造的。

# 颜色插件

# Color Kit

一款可基于指定颜色的快速生成指定梯度数量色阶的色卡的插件。

# Webgradients

一款提供 180 中渐变配色的插件。

# 设计

# Design Lint

一款检查设计中是否缺少文本、填充、笔触和效果样式的插件，它还会捕获不正确的边界半径值。适合在交付设计稿前运行检查设计细节。

# Wireframer

一款快速生成线框文本生占位符（SVG 图像）的插件，以构造高保真度的线框图 wireframe。

# Wireframe

一款提供大量线框图的插件。

# Hand Wireframe

将基本图像和文字（英文）转换为手绘风格。

# 交互

# Figmotion

一款制作原型交互的插件，操作详情可查看文档。

# 其他

# Mapsicle

一款交互式地图插件，使用 Mapbox 地图为您提供各种自定义选项，如地图样式修改和地点设置等功能而非简单的截图，可让您在设计原型中放置「动态」地图。
